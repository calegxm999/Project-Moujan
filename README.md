# Project-Moujan
The rubber ducky mouse project!


### What this is:
This is a project that will become available when it is complete, expected release is by 2030.
This will be similar to a rubber ducky but you can actually use it as a mouse, the idea of doing button combos to do specific actions was considered.
If enough people want it to use Ducky Script it will be implemented.
